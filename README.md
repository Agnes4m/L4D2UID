# L4D2UID

gsuid_core 插件，查询 L4D2 玩家数据（Anne 电信服）。

## 命令

- `l4查询 [SteamID]` — 查询玩家数据（历史 + 赛季），返回暗色主题图片
- `l4搜索 [昵称]` — 按昵称搜索玩家（文字返回）
- `l4绑定 [steamid32]` / `l4切换uid` / `l4删除uid` — 管理绑定 SteamID
- `l4切换 [电信/呆呆]` — 切换查询平台
- `l4状态` — 服务器状态 + 在线玩家列表，返回图片
- `l4统计` — 服务器荣誉殿堂，返回图片
- `l4帮助` — 帮助图片

所有命令返回图片，除 `l4搜索` 外。

## 要求

- Python >= 3.12
- `uv` 管理依赖，`ruff` 格式化 (line-length=120)

## 数据源

- `anne.trygek.com` — Anne 电信服 API

## 安装

```
core安装插件L4D2UID
core重启
```
